# Pitch Assets

This folder stores presentation collateral for PassportSOL.

Use it for:

- `.pptx` deck files
- exported slide images
- pitch scripts
- supporting presentation notes

Suggested convention:

- keep master decks at the top of this folder
- keep exported slide images in `docs/pitch/slides/`
- keep short scripts or speaking notes next to the deck they belong to

This keeps the repo root focused on product code instead of presentation files.
